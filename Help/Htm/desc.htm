HID Device Object

The HID Device object is a communication extension which can interact with input devices like keyboard, mouse, joystick which comply with the HID Standard.

It can from principle also communicate with all devices which are HID compatible, even Bluetooth devices on Windows 7 and 8.

Since HID Standard is supported natively on those platforms, there is no need to install drivers. This can lead to domotics application where the PC is controlling an electronic card.
